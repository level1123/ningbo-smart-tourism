# 宁波智慧文旅三维交互系统
本项目是浙江树人学院《人工智能》课程大作业。
## 项目简介
基于Cesium三维地图开发的浏览器端智慧文旅交互系统，支持宁波景点三维漫游、360°全景浏览、景点路线规划、AI导游问答、实时天气查询。
## 核心功能
1. 三维卫星文旅地图：标注多处文旅景点，自动生成游览路线
2. AIGC图标注释：调用SiliconFlowAPI用来AI生成图标注释，为画面增添趣味性
3. AI大模型交互导游：调用SiliconFlow和智谱AI大模型API，提供景点问答、游玩攻略生成
4. 实时气象模块：接入Open-Meteo免费API气象接口，展示景区天气
## 技术栈
前端：HTML/CSS/JavaScript、Cesium、Pannellum全景框架
AI工具：TRAE代码助手、AIGC文生图工具
## 运行方法
1. 克隆仓库至本地
2. VS Code安装Live Server插件，右键index.html启动本地服务
3. 浏览器自动加载三维文旅系统
